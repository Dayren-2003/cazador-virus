# 🦠 Cazador de Virus

## 📌 Objetivo
Eliminar virus antes de que infecten el sistema.

## 🎯 Justificación
Aplicar conocimientos de graficación 2D utilizando Canvas API.

## ⚙️ Funcionamiento
- Los virus aparecen aleatoriamente
- Caen hacia abajo con velocidad variable
- El jugador hace clic para eliminarlos
- Cada 10 puntos sube el nivel
- Si los virus llegan al fondo, se pierden vidas

## 🛠 Tecnologías
- HTML
- CSS
- JavaScript
- Canvas API
- Bootstrap
